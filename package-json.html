    Dashboard package.json

    Dashboard receives some configuration from merging its package.json with any modules and then finally your root package.json in that order..

    Your Dashboard server begins with a blank NodeJS project:

    $ mkdir project
$ cd project
$ npm init
$ npm install @userdashboard/dashboard

    After running npm init a package.json will be created in your project folder. Some Dashboard settings can be specified inside that package.json:

    • title will appear in the template.html header
    • additional links for the account or administrator menus
    • server request handlers to work with HTTP requests
    • HTML content handlers to work with content served to users
    "dashboard": {
  "modules": [
    "@userdashboard/stripe-subcriptions"
  ],
  "content": [
    "/src/modify-page-or-template-content.js"
  ],
  "server": [
    "/src/modify-requests-prior-to-completion.js"
  ],
  "menus": {
    "account": [
      {
        "href": "/account",
        "text": "My account",
        "object": "link"
      },
      {
        "href": "/account/signout",
        "text": "Sign out",
        "object": "link"
      }
    ],
    "administrator": [
      {
        "href": "/administrator",
        "text": "Dashboard administration",
        "object": "link"
      }
    ]
  }
}
