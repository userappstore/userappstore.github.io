    How to create reset codes

    Reset codes allow a user to sign in if they forget their password. You should verify the user's identify before doing this.

    • 1. Click "Dashboard administration"
    • 2. Click "Accounts"
    • 3. Click account to create reset code for
    • 4. Click "Create reset code"
    • 5. Enter new code and submit form
    • 6. Success
